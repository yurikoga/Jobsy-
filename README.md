# 💼 Jobsy - Freelance Your Way 🌟

**Jobsy** é uma **landing page interativa para freelancers**, inspirada no design do Figma com **vídeo animado**, cores modernas e layout clean. Ideal para mostrar plataformas de trabalho remoto, oportunidades e perfis profissionais.  

---

## ✨ Destaques do Projeto

- 🖥 **Vídeo Animado:** Animação do Figma que mostra freelancers em ação.  
- 🎨 **Design Limpo e Moderno:** Fundo claro, cores sóbrias e tipografia elegante.  
- 🌍 **Tema Freelancer:** Conteúdo voltado para profissionais independentes e projetos remotos.  
- 📱 **Responsivo:** Funciona perfeitamente em desktop, tablet e mobile.  
- 🎯 **Call-to-Action Eficiente:** Botões claros incentivando cadastro e exploração da plataforma.  

---

## 💻 Tecnologias Utilizadas

| Tecnologia | Uso |
|-----------|-----|
| HTML5     | Estrutura da página |
| CSS3      | Layout, cores e animações |
| JavaScript| Interações dinâmicas e efeitos |
| Vídeo Figma | Animação de freelancers para destaque visual |
